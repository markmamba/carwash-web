# Scorpio Carwash Web

React Router v7 (framework mode) admin portal for Scorpio Carwash. JavaScript only — no TypeScript.

## Stack

- React 19, JavaScript (`.js`/`.jsx`)
- React Router v7 framework mode (SSR)
- Vite 8
- Bootstrap 5 SCSS (react-bootstrap added in later phases)

See `.ai/instructions.md` for full coding conventions.

## Getting Started

```bash
npm install
npm run dev
```

Application runs at `http://localhost:5173`.

## Commands

```bash
npm run dev     # Development server with HMR
npm run build   # Production SSR build
npm run start   # Start production server
```
